# Ecommerce Project

* 技术选型
  * 前端： `react` `react-router` `redux` `antd` `axios`
  * 后端： `express` `mongoose`
  * 数据库： `mongodb`

* 文件结构
  参考 [Create React App with an Express Backend](https://daveceddia.com/create-react-app-express-backend/)

  为了能够让antd组件按需加载，执行了 `create-react-app` 中的 `eject` 命令，修改了babel的配置。
